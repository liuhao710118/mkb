## 系统操作

[uname命令](./uname命令.md)：用于查看系统内核信息

[hostnamectl命令](./hostnamectl命令.md)：用于管理和查看系统主机名（hostname）及主机相关信息的命令

## 进程相关

[top命令](./top命令.md)：最常用的性能监控工具之一，用于**实时显示系统中各个进程的运行情况**和系统整体的资源使用情况（CPU、内存、负载等)

[htop命令](./htop命令.md)：`htop` 是一个比 `top` 更增强、更美观的交互式系统监控工具

[iostat命令](./iostat命令.md)：用于监控 **CPU 使用率** 和 **磁盘 I/O 性能** 

[free命令](./free命令.md)：用于查看 **系统内存（RAM）和 Swap（交换分区）** 的使用情况

[ps命令](./ps命令.md)：用于显示系统中正在运行的进程信息 通常配合grep head命令使用



## 压缩与解压

[zip命令](./zip命令.md)：`zip` 和 `unzip` 是 Linux 中最常用的**压缩与解压工具**之一，兼容 Windows 的 `.zip` 格式。

[gzip命令](./gzip命令.md)：`gzip`是 Linux/Unix 系统中一个非常常用的**文件压缩和解压缩工具**

[tar命令](./tar命令.md)：`tar`（tape archive）命令最初用于将文件备份到磁带设备，现在主要用于**打包与压缩**文件和目录

## 基本文件目录操作

[ls命令](./ls命令.md)：列出当前目录下文件和子目录

[pwd命令]()：显示当前目录的路径

[cd命令]()：切换目录 

[tree命令](./tree命令.md)：用于以树状图形式列出目录的内容，直观展示目录结构。

[cp命令](./tree命令.md)：递归复制文件/文件夹保留属性

[rm命令](./rm命令.md)：删除文件/文件夹

[mkdir]()：创建文件夹



## 查看文件修改文件

[split命令](./split命令.md)：用于将大文件分割成多个小文件的工具，常应用于**分卷压缩、数据备份、日志切割、文件传输**

[touch命令](./touch命令.md)：创建空文件/更新文件的访问时间与修改时间

[cat命令](./cat命令.md)：查看文件内容,创建文件,将多个文件合并输出,输出内容到终端或重定向到文件

[less命令](./less命令.md)：是一个 **分页查看文件内容** 的工具

[head命令](./head命令.md)：用于查看文件的前若干行内容，默认显示前 **10 行**

[tail命令](./tail命令.md)：用于查看文件结尾部分的内容，默认显示 **最后 10 行**。常用于实时监控日志、查看文件末尾更新等

[vi编辑器](./vi编辑器.md)：`vi` 是 Unix/Linux 系统中最常用的文本编辑器之一，轻量、高效、几乎在所有系统中默认存在。
