## 一、基础认知（先建立正确世界观）

> 不然很容易被 K 线大师和群友带沟里 😅

### 1️⃣ 股票市场怎么运作

你要搞清楚这些概念：

- A 股 / 港股 / 美股 的区别
- 一级市场 vs 二级市场
- 交易规则
  - T+1 / T+0
  - 涨跌停
  - 交易时间
- 手续费、印花税、过户费

**目标**：
👉 知道钱是怎么亏的，不是“怎么就没了”。

------

### 2️⃣ 常见投资风格（非常重要）

你必须知道自己在玩哪一类游戏：

| 风格        | 本质        |
| ----------- | ----------- |
| 短线 / 打板 | 情绪 + 博弈 |
| 波段        | 趋势 + 技术 |
| 价值投资    | 公司经营    |
| 指数投资    | 经济增长    |
| 量化        | 统计 + 程序 |

⚠️ 程序员**最容易踩坑**的是：

> 学了点技术指标，就以为能短线暴富

------

## 二、技术分析（你会天然上手）

> 这是你最容易入门的部分，但**不要迷信**

### 你需要学：

- K 线基础（阳线、阴线、影线）
- 成交量（量价关系）
- 趋势 & 支撑 / 压力
- 常见指标（理解即可，不要全学）
  - MA
  - MACD
  - RSI

💡 程序员理解方式：

> 指标 = 特征工程
> 买卖点 = 信号触发
> 胜率 = 概率，不是确定性

------

## 三、基本面分析（拉开人与人的差距）

> 真正能长期赚钱的人，都绕不开这个

### 你要学会看：

#### 1️⃣ 财务三表

- 资产负债表（公司稳不稳）
- 利润表（赚不赚钱）
- 现金流量表（会不会死）

重点指标：

- ROE
- 毛利率
- 净利率
- 负债率
- 经营性现金流

#### 2️⃣ 行业与公司

- 公司靠什么赚钱？
- 行业是不是在增长？
- 护城河是什么？
- 上游 / 下游是谁？

📌 建议：
从 **自己懂的行业** 入手（IT / 软件 / 云 / AI / 半导体）

------

## 四、交易系统（程序员的主战场）

> 散户亏钱，80% 是因为 **没系统**

一个完整交易系统至少包含：

1. **买入条件**
2. **卖出条件**
3. **止损规则**
4. **仓位管理**
5. **复盘机制**

💬 用程序员的话说：

> 这是一个「有状态的策略系统」，不是拍脑袋 if-else

------

## 五、风险控制（比选股更重要）

如果你只记住一句话，记这句：

> **活着，才能继续交易**

必须掌握：

- 单笔最大亏损（如 ≤ 2%）
- 总仓位控制
- 不满仓
- 不梭哈
- 不补跌

程序员优势：
你知道 **爆仓 = 系统直接 exit**

------

## 六、心理与反人性（最难）

> 这里淘汰 90% 的人

你会遇到：

- 买了就跌，卖了就涨
- 盈利拿不住，亏损死扛
- 群友一喊就冲

解决方案：

- 用规则对抗情绪
- 交易前写下理由
- 交易后复盘

------

## 七、给程序员的进阶方向（强烈推荐）

如果你真的想把优势用到极致 👇

### 🔥 量化 / 数据化投资

你可以学：

- Python + pandas
- 回测框架（backtrader / zipline）
- 因子分析
- 简单策略（均线、动量）

⚠️ 注意：

> 量化 ≠ 印钞机
> 回测 ≠ 实盘

------

## 八、学习顺序（懒人版）

照这个顺序来，不容易翻车：

1. 市场规则 + 基础认知
2. 指数投资（先不亏钱）
3. 技术分析基础
4. 基本面
5. 交易系统
6. 量化 / 策略优化
