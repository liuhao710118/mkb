# 🆔 **id 命令详解**

`id` 用于 **显示当前用户或指定用户的 UID、GID 及所属的所有组**，是排查权限问题的常用工具。

------

# 🧩 **基本语法**

```bash
id [用户名]
```

- 不带参数 → 显示当前用户的身份信息
- 带用户名 → 显示指定用户的信息

------

# 📌 **输出内容说明**

示例：

```bash
id user1
```

输出：

```
uid=1001(user1) gid=1001(user1) groups=1001(user1),10(wheel),20(devops)
```

含义：

| 字段              | 说明                     |
| ----------------- | ------------------------ |
| `uid=1001(user1)` | 用户 ID（UID）和用户名   |
| `gid=1001(user1)` | 主组 ID（GID）和主组名   |
| `groups=...`      | 所属的所有组，包括附加组 |

------

# 🧪 **常用示例**

------

### ⭐ 1. 查看当前用户身份

```bash
id
```

------

### ⭐ 2. 查看指定用户身份

```bash
id alice
```

------

### ⭐ 3. 仅查看 UID

```bash
id -u user1
```

------

### ⭐ 4. 仅查看 GID

```bash
id -g user1
```

------

### ⭐ 5. 查看所属所有组（仅组 ID）

```bash
id -G user1
```

------

### ⭐ 6. 查看所属所有组（显示组名）

```bash
id -Gn user1
```

------

# 🔧 结合其他命令使用

### 检查文件权限时很常用：

```bash
ls -l file.txt
id user1
```

判断用户是否有读/写/执行权限。
